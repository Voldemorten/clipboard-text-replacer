Text replacement list

Use this page to add items to the text replacement list.

Items are replaced in the order they appear in the list.

Example: input text: /^(https?:\/\/(www\.)?[^/]+)/g replaced with localhost:3000 will replace any matching url with localhost:3000

Rules

The top rule is the most important. When matching on single rule, the first rule from the top that matches gets executed, and on chained rules are processed from bottom to top.
